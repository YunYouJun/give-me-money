- [Vue](https://vuejs.org)
- [Parcel](https://parceljs.org)
- [Vuex](https://vuex.vuejs.org)
- [iconfont](http://iconfont.cn)
- [Element](https://github.com/ElemeFE/element/)

```sh
# http://localhost:2333
```
